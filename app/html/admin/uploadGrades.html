Grades Instructions

Grade upload spreadsheet should be a CSV; any grades you upload will overwrite previous values for that deliverable.

The first row of the CSV should contain row headers. Any number of rows are allowed, but four are required: CSID, SNUM, GRADE, COMMENTS. Empty values are allowed for any GRADE or COMMENTS cell.
